# qr-code

This is QR-Code project created for frontendmentor.io

## To start project

1. Install HTTP server globally:

   ```sh
   npm install -g http-server
   ```

2. Start the server:

   ```sh
   http-server -p 8080 --host 0.0.0.0
   ```

3. Open in browser:

   ```
   http://127.0.0.1:8080
   ```
